Open-source orchestration

GraphClaw

A graph-based task orchestration system where humans and AI agents collaborate with explicit dependencies, scored priorities, and auditable lifecycle state.

Read Docs Backend Repository Cockpit Repository

Self-host first

Launches as a self-hosted stack. Start with the backend guide and cockpit guide.

Auditable by design

Release process uses signed artifacts, SBOM generation, and provenance attestation with maintainable governance docs.

Pluggable architecture

Database, gateway, LLM, and infra layers are swappable via interfaces so new providers can be added without rewriting core logic.

GraphClaw is released under Apache-2.0. For security disclosures, use the private advisory flow in the repository SECURITY policy.
